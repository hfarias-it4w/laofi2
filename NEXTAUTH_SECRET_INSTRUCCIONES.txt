# NextAuth Secret
# Genera una clave secreta segura para NextAuth.js
# Puedes usar este comando en tu terminal:
#
#   openssl rand -base64 32
#
# Copia el resultado y pégalo en tu archivo .env.local así:
#
# NEXTAUTH_SECRET=TU_CLAVE_GENERADA
#
# Ejemplo:
# NEXTAUTH_SECRET=Q2FzYXNhbG9zQ29kaWdvczEyMzQ1Njc4OTAxMjM0NTY3ODkwMTIzNDU2Nzg5MA==
#
# Luego, reinicia tu servidor Next.js y vuelve a iniciar sesión en la app.
